# Fruit Store 음악 모듈

## 개요
초등 수업에서 교사가 사용하는 음악 모듈입니다. 과일과 음표 길이를 연결해 학습하는 인터랙티브 활동을 제공합니다.

## 로컬 실행 방법
1) 터미널에서 이 폴더로 이동
2) 아래 명령 실행

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000/` 로 접속합니다.

## 정적 배포
GitHub Pages 기준으로 상대경로(`./`)를 사용합니다.

## 파일 구조
- `index.html`
- `style.css`
- `script.js`
- `assets/`
